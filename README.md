## Descripción

Aplicación que estoy desarrollando para facilitar el manejo de archivos de guardado de videojuegos, permitiendo la **coexistencia de múltiples partidas** o "Perfiles" dentro de un mismo juego y guardando un **historial de checkpoints** para poder volver a un estado previo.
 

## Módulos
El programa va a tener un modulo "**Servidor**" que se va a encargar de manejar la base de datos con las partidas guardadas, y un modulo "**Cliente**" que va a permitir conectarse al modulo servidor local para cargar y descargar archivos de guardado según indicado por el usuario, o conectarse con módulos Servidor externos para obtener archivos de guardado de otros nodos, facilitando la transferencia de archivos entre computadoras para facilitar la continuidad al jugar una misma partida a través de distintos dispositivos.
